Sass

Für dieses Projekt wurde CSS in Verbindung mit Sass benutzt. Der Header ist dabei inspiriert von dem Layout der Seite, die wir in der Vorlesung erstellt haben.

Das geschriebene CSS zeichnet sich durch folgende Eigenschaften aus:

  • Die Datei style.css (bzw. style.scss) enthält das grundlegende Design für die Website. Das umfasst die Farbpalette, den Header/Footer, die Sections etc.
  • Jede HTML-Seite besitzt zudem eine weitere CSS-Datei (bzw. SCSS-Datei), in welcher das Styling von Komponenten, die exklusiv in dieser Datei vorkommen (wie z.B: das Canvas auf der "Unterschreiben"-Seite), definiert ist
  • Das CSS für bestimmte Strukturen, die mehrfach vorkommen (bsp. Pop-up) wurde ebenfalls in eigene Dateien ausgelagert, um Redundanz zu verhindern und gleichzeitig die style.scss übersichtlich zu halten
  • Das "Theme" der Website ist leicht veränderbar, indem nahezu alle CSS-Parameterwerte in einzelne Dateien ausgelagert wurden, die somit als eine Art Konfigurationsdatei dienen. Dafür existieren folgende Dateien:
    • theme beschreibt die Farbpalette der Website. In ihr kann beispielsweise die Akzent- und Hintergrundfarbe der Seite mit jeweils nur einer Variable verändert werden
    • dimens beihaltet sämtliche Größen und Abmessungen wie px, % oder rem. Auch die Breakpoints für das Responsive Design sind hier spezifiziert
